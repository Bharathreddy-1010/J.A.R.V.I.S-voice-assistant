# J.A.R.V.I.S Voice Assistant

A personal AI assistant inspired by JARVIS from Iron Man.  
Built using Python, Speech Recognition, and ADB for automated tasks.

## Features
- Voice command recognition
- Message automation
- Open websites and apps
- JARVIS-style animations

## Installation
1. Clone the repository:
   git clone https://github.com/Bharathreddy-1010/J.A.R.V.I.S-voice-assistant.git
2. Navigate to folder:
   cd J.A.R.V.I.S-voice-assistant
3. Install dependencies:
   pip install -r requirements.txt

## Usage
Run the assistant:
python jarvis.py

Open the UI:
open index.html
